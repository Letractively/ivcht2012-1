Структура проекта.

Src
  main

    java -- Содержит сервлеты и классы
      controller -- Папка содержит сервлеты, отвечающие за вызов страниц JSP, с передачей в них определенных данных
        *DatabaseCreationController --- Контроллер-сервлет, создающий базу данных и таблицы в ней

      dao        -- Папка содержит классы и интерфейсы, взаимодействующие с БД,
                                                    т.е. классы соответствующие таблицам базы данных,
                                                    с методами работающими с данными в БД, а также другие классы
                                                    позволяющие или помогающие работать с БД
        *ConnectionDBDao --- Класс подключения к базе данных
        *MessagesDBDao   --- Класс, реализующий работу с таблцией сообщений в базе данных
        !TopicsDBDao     --- Класс, реализующий работу с таблцией тем в базе данных через SQL запросы


      model      -- Папка содержит общие классы сущностей
        !Topic           --- Класс темы, с полями и конструкторами
        *Message         --- Класс сообщения

      service    -- Папка содержит классы, выполняющие некоторые обработки данных или расчеты. Например глобальные
                                            операции с базой данных (создание, удаление, экспорт, импорт) и прочее.
        *DatabaseService --- Класс обслуживания базы данных
        !TopicsDaoService--- Класс-служба, выполняющий все необходимые для сайта операции - получения и расчеты данных

      utils      -- Папка содержит вспомогательные классы, призванные выполнять мелкие функции или упрощения операций
                                                        и какие обертки?
        !ServiceInstancer--- Служит для получения экземпляров service классов, через обработки которых получается доступ
                                                                             к DAO классам обращения к данным

    --------------------------------------------------------------------------------------------------------------------
    Controller/Servlet -> ServiceInstancer -> ITopicsService -> ITopicsDao -> Topic+DB
    --------------------------------------------------------------------------------------------------------------------
    recources -- dunno

    webapp -- JSP/HTML наполнение
      recources        -- Рерурсы испльзуемые JSP/HTML страницами
        css               -- Папка содержащая CSS стили
        img               -- Папка содержащая используемые изображения
      WEB-INF->web.xml -- Файл задающий соответствия между сервлетами и JSP/HTML страницами
      index.jsp        -- Welcome page
      about.jsp        -- About project/team page
      topicList.jsp    -- List of Topics page
      topicAdd.jsp     -- Add Topic page
      topicEdit.jsp    -- Edit Topic page
      topicSearchResult.jsp -- Search Topics result
      messageList.jsp  -- List of Messages page
      messageAdd.jsp   -- Add Message page
      messageEdit.jsp  -- Edit Message page
      messageSearchResult.jsp -- Search Message result
